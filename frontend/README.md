# TRUEPLOT Frontend

Next.js application for the TRUEPLOT MVP local development foundation.

Run locally:

```bash
npm install
npm run dev
```

Open:

```text
http://localhost:3000
```

Routes:

- `/` dashboard
- `/properties` listings
- `/properties/new` create listing
- `/properties/[id]` detail workspace with document upload and AI summaries
- `/agents` verified agent listings
- `/agents/new` agent onboarding
- `/agents/[id]` agent detail and approval
- `/admin` moderation actions
