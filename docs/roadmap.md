# Roadmap

The current repository is a local MVP. The following roadmap outlines likely future phases.

## Phase 1: OCR and Document Intelligence

- OCR for EC, 1B, Adangal, sale deed, Pattadar passbook, and FMB map
- Structured document extraction
- Survey number and party-name extraction
- Basic mismatch detection across documents

## Phase 2: Telugu AI Improvements

- Better Telugu legal summarization
- Bilingual risk explanation
- AP-specific land record terminology
- Human review feedback loops

## Phase 3: Authentication and Roles

- Buyer, seller, agent, admin roles
- Session management
- Secure file access
- Audit logs for review actions

## Phase 4: Payments

- Listing fees
- Agent service fees
- Lease workflow payments
- Transaction escrow exploration

## Phase 5: Leasing Intelligence

- Lease pricing benchmarks
- Lease term comparison
- Deposit and escalation tracking
- Commercial lease workflow support

## Phase 5b: Fair Market Value Intelligence

- Better pricing signals for sale and lease listings
- District, mandal, and village market context
- Road and highway proximity enrichment
- Confidence scoring refinement
- Human review feedback for pricing signals

## Phase 6: GIS Integrations

- Map-based property search
- Survey boundary overlays
- FMB map visualization
- Location risk indicators

## Phase 7: Bank Integrations

- Loan eligibility support
- Property document checklist for banks
- Valuation handoff
- Lien and encumbrance workflows

## Phase 8: Government and Registry Integrations

- Official land record APIs where available
- Encumbrance certificate verification workflows
- Mutation and revenue-record checks
- Compliance review with legal experts

## Phase 9: Production Infrastructure

- Alembic migrations
- Production Docker images
- Kubernetes deployment
- CI/CD pipelines
- Secrets management
- Database backups

## Phase 10: Observability

- Structured logging
- Metrics and dashboards
- Error tracking
- Audit trails for document and verification events

## Phase 11: Mobile Apps

- Buyer app
- Seller app
- Agent field workflow app
- Document capture and upload from mobile
