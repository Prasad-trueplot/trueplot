# Founder Demo Script

Use this script for co-founder, advisor, or investor walkthroughs of the local TRUEPLOT MVP.

## 1. TRUEPLOT Problem Statement

TRUEPLOT addresses the friction in Andhra Pradesh land and property transactions. Buyers, sellers, and agents usually deal with fragmented records, unclear verification, delayed review, and too much manual coordination.

## 2. Market Gap

The market lacks a simple platform that combines:

- Verified property listings
- Document review workflows
- AI-assisted legal summaries
- AI-assisted pricing intelligence
- Verified agent assignment
- Lightweight admin moderation

## 3. Product Positioning

TRUEPLOT is a verified real estate transaction platform, not a generic classifieds app. The product is designed around trust, intelligence, and local workflow clarity for AP land and property use cases.

## 4. Live Demo Steps

1. Open `http://localhost:3000`.
2. Log in as the seeded demo user for the role you want to show.
3. Open the property listings page.
4. Open a property workspace.
5. Show the document vault, AI legal summary panel, and pricing intelligence panel.
6. Open the admin page to show moderation.
7. Open the agents page to show verified agent workflows.

## 5. AI Legal Summary Walkthrough

1. Open an uploaded EC or 1B document.
2. Generate the AI legal summary.
3. Show the English and Telugu summary outputs.
4. Point out the ownership summary, document insights, risk flags, and recommended next steps.
5. Emphasize the disclaimer that this is AI-assisted review content, not legal advice.

## 6. AI Pricing Intelligence Walkthrough

1. In the same property workspace, open the pricing intelligence panel.
2. Click Generate price estimate.
3. Show the sale or lease estimate range.
4. Call out the confidence score as a planning signal, not a valuation guarantee.
5. Walk through the pricing factors, district influence, mandal influence, village influence, and road or highway placeholder.
6. Read the AI-assisted valuation disclaimer aloud.

## 7. Verified Agent Workflow

1. Open the verified agents page.
2. Show an agent profile with district, mandal, village, leasing, and NRI specialization.
3. Explain that the admin approves the agent before assignment.
4. Return to a property workspace and assign the verified agent.
5. Show the assigned agent details on the property.

## 8. Admin Approval Workflow

1. Open `http://localhost:3000/admin`.
2. Show pending listings, pending agents, and moderation actions.
3. Approve the property listing.
4. Mark the listing as verified.
5. Approve a verified agent profile.

## 9. Revenue Model

TRUEPLOT can monetize through:

- Listing subscriptions or posting fees
- Verified agent onboarding and premium placement
- Premium legal and document intelligence
- Lease workflow support
- Enterprise or broker tooling later

This MVP does not implement payments yet. The model is a placeholder for discussion.

## 10. 6-Month Roadmap

1. OCR for EC, 1B, Adangal, and sale deed files.
2. Better Telugu AI explanations.
3. Stronger authentication and permissions.
4. Pricing intelligence refinement.
5. Leasing intelligence and broker tools.
6. GIS and map integrations.

## 11. Next Founder Decisions

The next decisions to make are:

- Which AP district or market to focus on first
- Whether the first paid workflow should be listings, agent tools, or legal review
- How much manual review should stay in the loop
- Whether the first product wedge is buyer trust, seller workflow, or agent efficiency
- Which data sources to prioritize for OCR and location intelligence
